# YB Photography & Graphics

A modern, responsive multi-page website for a photography and graphics studio, built using HTML, CSS, and JavaScript for Frontend and PostgreSQL for database. This project currently showcases a complete frontend solution with dynamic UI components, interactive features, and clean design.

---

##  Live Demo

🔗 https://tanviepari.github.io/YB-Photography-And-Graphics/

---

##  Features

*  **Dynamic Portfolio Section**

  * Category-based filtering (Wedding, Pre-Wedding, Portrait, Events, Graphics)
  * Images rendered dynamically using JavaScript
  * Lightbox view for full-screen image preview

*  **Booking Modal**

  * Popup form with background blur
  * Click outside / close button functionality
  * Form validation

*  **Contact Form**

  * Clean UI with validation
  * Structured for backend integration

*  **Feedback Form**

  * Interactive star rating system
  * Client feedback submission

*  **Pricing Page**

  * Structured packages with highlighted “Most Popular” plan

*  **About Page**

  * Story, mission, and values sections
  * Reusable components for consistency

*  **Fully Responsive Design**

  * Mobile-friendly navigation (hamburger menu)
  * Adaptive layouts for all screen sizes

## Database

A PostgreSQL database has been designed for this project.
It includes tables for:
- Contact
- Booking
- Testimonials
- Gallery
  
The database schema is provided in the "YB-Studio-Database.sql" file.
